Calitatea aerului este buna iar poluarea aerului prezintă un risc redus. Calitatea aerului este medie.Valorile anumitor poluanti pot fi dadunatoare pentru oamenii cu probleme de sanatate. Nivelul ridicat de poluanti din aer poate provoca boli respiratorii la persoanele cu expunere prelungită.
